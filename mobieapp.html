Medical Cost

  • Flask API

Project about designing and creating mobile application for medical cost to represent data analysis by connecting to the database using MongoDB.
The result are shown in the pictures below...

The first tab allows users to insert and find data from database.

The second tab shows data visualization comparing the factors (bmi, age) in terms of Line Charts, Bar Charts and Doughnut Charts.

The third tab shows data overview for example average value, minimun value and maximum value of data.
